## 🎯 How to Use qr-kit

### Create a QR Code

1. Enter the text or website link you want your QR code to show.
2. Click the ‘Generate’ button.
3. Your QR code will appear instantly on the screen.

### Add a Logo

qr-kit lets you put a small logo on your QR code. To do this:

1. Click ‘Add Logo’.
2. Select an image file from your computer (PNG or JPG).
3. The logo will appear in the center of the QR code.

This helps personalize your code while keeping it easy to scan.

### Export as PDF

To save your QR code for printing or sharing:

1. Click ‘Export’.
2. Choose the PDF option.
3. Pick a location on your computer to save the file.
4. Click ‘Save’.

The PDF file contains your QR code in high quality.

### Optimize Links

If your QR code points to a long URL, use the link optimizer feature:

1. Click ‘Optimize Link’.
2. Paste your long URL.
3. qr-kit will create a shorter, clean link.
4. Use this short link for your QR code instead.

Short links are easier for people to type or remember.

## 🛠 Troubleshooting

If qr-kit does not open or work correctly:

- Make sure your Windows system is updated.
- Check if any antivirus software blocked the app. Allow access if needed.
- Try running the program as administrator (right-click the icon, choose ‘Run as administrator’).
- Restart your computer and try again.

If QR codes do not scan:

- Ensure the logo is not covering too much of the code.
- Make sure the screen or print is clear and flat.
- Try black and white QR codes for better scanning.
